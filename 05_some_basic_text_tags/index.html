Some basic html text tags

Center tag:

CENTER TAG aligns the contents to the center of the web page. It was deprecated from the previous versions of HTML. So we use css properties to align the things.

Paragraph text:

The <p> element marks all the text inside it as a distinct paragraph text. The content which is encapsulated with p tag, is rendered as a paragraph text.

Headings:

Headings are like titles, but they are actually displayed on the page. Html provides six levels of headings, and the corresponding elements are: <h1>, <h2>, <h3>, <h4>, <h5>, and <h6>. The higher the number, the less prominent the heading. H1 is the main heading and the remaining are the subheadings.

Character Entities:

HTML character entities are basically a set of characters (entity) used to represent a few characters reserved by the HTML, especially invisible characters or characters difficult to type out using a regular keyboard.

Address tag:

The address tag is used to define contact information for the author or owner of a document or an article. Typically, it is displayed in italics and may include information such as author's name, an email address, phone number, or physical address & even links to particular pages.

Cite tag:

The cite tag in HTML is not a formatting tag; it is a semantic tag. The cite tag is used to define the title of a creative work, such as the title of a book, movie, song, or a reference to the name of a creative work or the name of its author.
