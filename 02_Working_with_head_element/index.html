Inside the head element, there are 6 key child elements are there. title tag - gives us a title for the page and is often displayed in the title bar of the window (tab). meta element allows us to provide metadata about the document that can be anything from the viewport and how it can be render or it can be keywords and information that can be used in searches or to index the pages. script element - allows us to provide either right inside the document or linking to another document some script or code that can make our pages more interactive. style element - allows us to provide inline styling. Styles are additional information about how to render ertain content. link element - can provide links to a variety of different types of resources that are related to our page. External styling (CSS) is provided through linking. base element - allows us to specify a base address with which all our links to other documents will be based on.
